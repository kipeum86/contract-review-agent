## AI Limitations — Hallucinations and Errors

This project is powered by a large language model (Claude). LLMs can and do produce **hallucinations, factual errors, and incorrect legal analysis**. Outputs may:

- Misidentify or misclassify contract clauses
- Overlook material risks or obligations
- Generate legally incorrect suggestions
- Fabricate references to statutes, case law, or contract provisions that do not exist

**A qualified human must always review and verify every output** before any reliance or action. The agent recommends; the human decides.

## Data Security and Privacy

All processing runs **locally on your filesystem** — no external servers or databases are involved. However, the agent makes LLM API calls, which means **contract text is sent to the model provider** (Anthropic).

If you intend to use this for real legal work with confidential documents:

- **At minimum**, use a **commercial API key** (not a free-tier or consumer plan) so that standard commercial data handling terms apply.
- **Ideally**, use an **Enterprise plan with a Zero Data Retention (ZDR) agreement** — this ensures your data is not used for model training and is not retained on Anthropic's servers beyond the request lifecycle.
- **Always** check with your organization's **security, compliance, and IT departments** before processing any confidential, privileged, or personally identifiable information through any AI tool.
- Review [Anthropic's data usage policies](https://www.anthropic.com) for the most current terms applicable to your plan.
