# Contract Drafting (Roadmap — v2)

This feature is on the v2 roadmap and is not yet fully implemented.

$ARGUMENTS

---

## When available, this command will:

1. **Intake**: Accept the user's drafting request (contract type, parties, key terms)
2. **Interview**: If details are insufficient, conduct a structured interview (max 10 rounds) to gather: parties, business context, core terms, risk posture, legal preferences, special provisions
3. **Template lookup**: Search the library for matching templates and select clause tiers based on negotiation leverage
4. **Generate**: Draft the contract using template clauses or from scratch if no template exists
5. **Self-review**: Automatically check for completeness, internal consistency, placeholders, and risk issues
6. **DOCX output**: Produce a signing-ready DOCX with professional legal formatting

## For now

You can still ask for help drafting contracts using natural language. The agent will assist based on general legal knowledge and any templates available in the library, but the full structured pipeline (interview, self-review, round management) is not yet in place.
